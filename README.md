# ProductAndSalesDisplay

To start the server, follow these steps:

- In your console make sure you are in the .csproj folder location i.e `cd path\to\source\repos\ProductAndSalesDisplay\ProductAndSalesDisplay`
- Run the following command to start the application:
  ```Powershell
	dotnet run
  ```
- Server should be running on `http://localhost:5250/` as set in launchSettings.json

To start the client, follow these steps:
- Open a new console window and navigate to the FrontEnd folder i.e `cd path\to\source\repos\ProductAndSalesDisplay\FrontEnd`
- Run the following command to start the application:
  ```Powershell
	npm run dev
  ```
- Client should be running on `http://localhost:5173/` as set in vite.config.js

To run backend tests, follow these steps:
- In your console make sure you are in the .csproj folder location i.e `cd path\to\source\repos\ProductAndSalesDisplay\ProductAndSalesDisplay`
- Run the following command to execute the tests:
  ```Powershell
	dotnet test
  ```

To run frontend tests, follow these steps:
- In your console make sure you are in the FrontEnd folder location i.e `cd path\to\source\repos\ProductAndSalesDisplay\FrontEnd`
- Run the following command to execute the tests:
  ```Powershell
	npm test
  ```

Should be good to go! Enjoy :)
